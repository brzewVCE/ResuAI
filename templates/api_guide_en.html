Google Gemini API Key Setup Guide

Follow these steps to get your free API key for ResuAI

1 Go to Google AI Studio

Visit the Google AI Studio website to create your API key:

https://aistudio.google.com/app/apikey

You'll need to sign in with your Google account if you haven't already.

2 Create a New API Key

Once you're on the API Keys page:

  • Click the "Create API Key" button
  • Choose "Create API key in new project" (recommended)
  • Give your project a meaningful name like "ResuAI"
Screenshot: API key creation interface
3 Copy Your API Key

After creating the API key:

  • Your new API key will be displayed
  • Click the "Copy" button to copy it to your clipboard
  • The key will look something like this:
AIzaSyB***********************************

5 Free Tier Information

Google Gemini API offers a generous free tier:

  • 15 requests per minute
  • 1,500 requests per day
  • 1 million tokens per month

This is more than enough for typical resume building activities. You can monitor your usage in the Google AI Studio dashboard.
